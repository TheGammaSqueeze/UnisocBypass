# 027 - CPU 2200 MHz (step up from 2150)

## Purpose

Exp 026 (2150 MHz) stable at 1050 mV. Step +50 MHz more.

If this crashes or throttles hard, 2150 is the per-chip CPU ceiling at
stock voltage. If stable, consider 2250 MHz next.

## DT change vs exp 026

CPU top OPP: 2150 -> 2200 MHz. Everything else unchanged.
